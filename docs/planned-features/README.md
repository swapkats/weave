# Planned Features

This directory contains documentation for features that are **planned but not yet implemented** in Weave.

## Status: Not Implemented

The features documented here are part of the roadmap for future versions of Weave. They are included for design discussion and planning purposes.

**⚠️ WARNING**: Do not attempt to use these features as they are not available in the current version.

## Planned Features

- **Cloud Deployment** (`cloud-deployment.md`) - AWS Lambda, GCP Cloud Functions, Docker deployment
- **Advanced Deployment** (`deployment.md`) - Provider config, infrastructure management, deployment strategies

## Implementation Timeline

These features are planned for:
- **v2.0**: Basic deployment support, provider configurations
- **v3.0**: Advanced deployment strategies, full cloud integration

See the main [ROADMAP.md](/docs/ROADMAP.md) for the complete feature roadmap and implementation timeline.

## Contributing

If you're interested in implementing any of these features, please:
1. Review the design documentation in this directory
2. Check the main ROADMAP.md for current status
3. Open an issue to discuss the implementation approach
4. Submit a PR with your implementation
