# Research Assistant Example

AI-powered research workflow for comprehensive analysis and reporting.

## Workflow

```
Literature Review → Data Collection → Synthesis → Peer Review
```

## Agents

1. **Literature Reviewer** - Reviews academic papers and studies
2. **Data Collector** - Gathers statistics and evidence
3. **Synthesizer** - Combines findings into coherent report
4. **Peer Reviewer** - Validates quality and accuracy

## Usage

```bash
# Full research workflow
weave apply

# Quick research (skip peer review)
weave apply quick_research

# Custom research topic
weave apply --input "Impact of AI on education"
```

## Use Cases

- Academic research
- Market analysis
- Competitive intelligence
- Due diligence
- Policy research

## Output

Generated reports include:
- Literature review summary
- Data and statistics
- Synthesis and insights
- Citations and sources
